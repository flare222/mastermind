Mastermind

Instructions

You have ten chances to try to correctly decode the colour combination and position.

Select four colours to make a guess. You will receive one black peg for each correct colour and position, one grey peg for each correct colour, but incorrect position and one red peg for each guess where neither the colour nor position was correct.

Duplicate colours are allowed, but not blanks.
